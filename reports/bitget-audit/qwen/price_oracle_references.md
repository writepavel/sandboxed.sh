# Price Oracle Technical References

Below are the key sources I referenced when compiling the technical overview of price oracle systems:

1. **Euler Docs - Price Oracles**
   - URL: https://docs.euler.finance/concepts/core/price-oracles/
   - Notes: Explains how price oracles are used in the Euler protocol, highlighting the dynamic integration of various pricing sources for different assets.

2. **Oracle Retail Pricing Cloud Service Architecture**
   - URL: https://docs.oracle.com/cd/E99185_01/rpcs/pdf/160029/html/operations_guide/rpcs-og-architecture.htm
   - Notes: Technical architecture overview of Oracle's retail pricing cloud service.

3. **Technical Guide: Build a Price Oracle - Medium**
   - URL: https://medium.com/@Chainsight_Network/technical-guide-build-a-price-oracle-9f9d7ee4a655
   - Notes: Explains how to build a price oracle and how they can be used for token price reference in trading systems.

4. **Sei Docs - Oracle System Technical Overview**
   - URL: https://docs.sei.io/learn/oracles
   - Notes: Describes Sei's native oracle module that provides decentralized, secure price data on-chain.

5. **Maker Protocol Technical Docs - Oracle Module**
   - URL: https://docs.makerdao.com/smart-contract-modules/oracle-module
   - Notes: Explains how the Maker oracle module provides price data for different collateral types.

6. **IACR Paper on Distributed Blockchain Price Oracle**
   - URL: https://eprint.iacr.org/2022/603.pdf
   - Notes: Academic paper on the design and security considerations for distributed price oracles.

7. **GitHub - euler-xyz/euler-price-oracle**
   - URL: https://github.com/euler-xyz/euler-price-oracle
   - Notes: Open-source implementation of price oracle adapters compatible with ERC-7726.

8. **Cyfrin - Price Oracle Manipulation Attacks Guide**
   - URL: https://www.cyfrin.io/blog/price-oracle-manipulation-attacks-with-examples
   - Notes: Comprehensive guide to price oracle manipulation attacks and mitigation strategies.

9. **Chainlink Documentation - Architecture Overview**
   - URL: https://documentation-private-puce.vercel.app/docs/architecture-overview/
   - Notes: Official Chainlink documentation describing how API requests are handled by oracle nodes.

10. **Chainlink Documentation - Data Feeds Architecture**
    - URL: https://docs.chain.link/architecture-overview/architecture-overview
    - Notes: Details on how Chainlink data feeds bring off-chain data on-chain for smart contracts.

11. **Cyfrin - Chainlink Data Feeds Overview**
    - URL: https://updraft.cyfrin.io/courses/chainlink-fundamentals/oracles-and-chainlink-data-feeds/chainlink-oracles-overview
    - Notes: Overview of Chainlink as a decentralized oracle network and its data feed implementation.

12. **Medium - Oracle Problem in DeFi: Chainlink and Price Feed Architecture**
    - URL: https://medium.com/@arunpandeycdac/the-oracle-problem-in-defi-chainlink-and-price-feed-architecture-29cf1dd31342
    - Notes: Discusses the "oracle problem" in DeFi and Chainlink's solution with decentralized oracles.

13. **RareSkills - How Chainlink Price Feeds Work**
    - URL: https://rareskills.io/post/chainlink-price-feed-contract
    - Notes: Technical explanation of Chainlink price oracles, their smart contract implementation, and the price update process.
